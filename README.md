# abng_portfolioone

يصلح لعرض الشركات واخبارها ومجلس ادارتها والاعمال والمشاريع مع امكانية التطوير السريع في الموقع بفضل نظام ادارة المحتوى backdrop cms المعروفة
مازال القالب في مراحله الاولى وتحت التطوير
